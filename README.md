# Climate Resilience Monitoring Dashboard

A web-based geospatial dashboard for real-time and time-series monitoring of climate hazards using satellite data, cloud-based processing, and interactive Web GIS.

## Motivation
Governments and agencies worldwide are investing in climate resilience and disaster risk reduction. This project demonstrates how Earth Observation data can be transformed into an accessible decision-support tool for monitoring environmental risks such as floods, droughts, and vegetation stress.

## Key Features
- Real-time and near–real-time environmental monitoring
- 12-month time-series analysis for selected regions
- Cloud-based satellite data processing using Google Earth Engine
- Interactive web dashboard designed for non-technical decision-makers

## Data & Technologies
- **Satellite data**: Sentinel-1 (SAR), Sentinel-2 (Optical)
- **Cloud processing**: Google Earth Engine (Python API)
- **Analysis**: Time-series analysis of rainfall and vegetation indices
- **Web GIS**: Streamlit / Folium
